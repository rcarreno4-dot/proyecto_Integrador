# Proyecto Integrador (MVC)

Repositorio organizado con **solo código fuente** del programa SIGPRA en estructura tipo MVC.

## Estructura

- `src/controller`: clases de arranque/navegación principal.
- `src/data`: acceso a datos (DAO, conexión y consultas).
- `src/modelo`: entidades y tipos de dominio.
- `src/view`: vistas Swing (frames/paneles) y recursos de interfaz.
- `src/config`: configuración de base de datos.
- `src/session`: sesión de usuario y modo de ejecución.
- `src/db.properties`: parámetros de conexión Oracle.

## Nota

Se excluyó documentación, entregables, zips y archivos temporales para mantener este repositorio enfocado únicamente en código.
